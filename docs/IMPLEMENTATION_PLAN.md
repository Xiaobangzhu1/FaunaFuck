# IMPLEMENTATION_PLAN - 仓库整理实施计划

## 1. 与 PRD 对齐说明
- 本计划仅覆盖“目录/文件命名整理 + 说明文档”，不触碰运行逻辑。
- 目标是结构清晰、命名一致、引用更新完整。
- 当前补充任务是“修复 `cli` 启动链路”，只做最小必要改动。
- 本轮收尾目标：完成当前未勾选的 5 项任务（方向属性、方向渲染、颜色可见性验证）。
- 本轮新增目标：将方向可视化从三角形恢复为箭头叠加，并新增最近 5 分钟日志调取能力。
- 维护方式：优先在原有计划基础上追加任务或更新勾选状态，不重写整份计划。
