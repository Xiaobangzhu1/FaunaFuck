# DNA_RULES - DNA 转录规则（MODE 2）

## 1. 规则状态
- 本项目仅使用 `MODE == 2`。
- `MODE == 1` 已废弃，不再使用。

## 2. DNA 碱基集
- `!`
- `?`
- `>`
- `<`

## 3. 转录映射（DNA -> RNA）
```python
translate_dict = {
    '!!': '+', '!?': ',', '!>': 'd', '!<': 'a',
    '?!': '.', '??': '-', '?>': 'w', '?<': 's',
    '>!': '[', '>?': '}', '>>': '>', '><': 'e',
    '<!': ']', '<?': '{', '<<': '<', '<>': 'b',
}
```

## 4. 打开方式（交互页）
- 命令：`dna`
- 快捷键：`/`
- 按钮：右侧交互区独立按钮

## 5. 关闭方式
- `ESC`
- 浮窗右上角关闭按钮
