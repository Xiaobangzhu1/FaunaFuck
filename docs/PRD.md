# PRD - FaunaFuck 产品需求文档（高层）

## 1. 文档定位

- 本文档仅定义：项目做什么、为什么做、用户看到什么、成功标准。
- 细节规则主权在系统设计文档；若与 PRD 冲突，先标记 `待确认`，不得直接按任一方实现。
- 实施顺序与任务拆解由 `docs/IMPLEMENTATION_PLAN.md` 负责。

## 2. 项目概述

FaunaFuck 是一个类 Brainfuck 的细胞自动机模拟器。系统通过 DNA→RNA 转录驱动细胞行为，支持在生态箱中观察群体动态、规则干预与运行反馈。

## 3. 目标

- 提供可交互的生态箱运行环境（投放、运行控制、规则调整、日志反馈）。
- 支持基于 DNA/RNA 规则的细胞行为演化观察。
- 保持主要交互流程可理解、可验证、可复现。
- 提供仓库级 `README.md` 作为首次进入项目的说明入口。

## 3. 核心规则

### 3.1 基因系统
- **DNA碱基**: `!`, `?`, `>`, `<`（仅 `MODE == 2`）
- **转录规则** (DNA → RNA, MODE 2 only):
  - `!! -> +`, `!? -> ,`, `!> -> d`, `!< -> a`
  - `?! -> .`, `?? -> -`, `?> -> w`, `?< -> s`
  - `>! -> [`, `>? -> }`, `>> -> >`, `>< -> e`
  - `<! -> ]`, `<? -> {`, `<< -> <`, `<> -> b`
- `MODE == 1` 视为废弃，不再作为可选转录模式。

### 3.2 RNA指令集
| 指令 | 功能 |
|------|------|
| d/a/w/s | 移动（右/左/上/下） |
| +/- | 增加/减少当前位置神经递质 |
| , | 自我复制（繁殖） |
| . | 自杀 |
| [n | 若当前channel值为0则跳转到n |
| ]n | 若当前channel值>0则跳转到n |
| > / < | 切换channel |
| { / } | 保留控制符（当前执行器读到后结束本帧） |

### 3.3 细胞生命周期
- 新生细胞有一帧无敌期 (`locked`)
- 死亡条件: `die_mode` (周围细胞数阈值) 或 `.` 指令
- 繁殖: `,` 指令触发，子代DNA可能突变

### 3.4 神经递质系统 (NTs)
- 256通道三维矩阵 (width × height × channels)
- 每通道值范围 0-255 (uint8)
- 细胞可读写当前位置的channel值用于条件跳转

## 4. 配置参数

### 4.1 地图配置 (MapConfig)
| 参数 | 默认值 | 说明 |
|------|--------|------|
| width | 100 | 地图宽度 |
| height | 100 | 地图高度 |
| channels | 256 | 神经递质通道数 |
| max_instructions | 1000 | 每细胞每帧最大指令数 |

### 4.2 细胞配置 (CellConfig)
| 参数 | 默认值 | 说明 |
|------|--------|------|
| gene_DNA | "<>!?!?><" | 初始基因序列 |
| original_num | 10 | 初始细胞数量 |
| die_mode | 3 | 周围细胞数死亡阈值 |
| base_mutation_rate | 1e-3 | 碱基突变概率 |
| gene_mutation_rate | 1e-4 | 基因突变概率 |
| ribosome_loop | False | 核糖体循环模式 |
| pure_mode | False | 纯净模式(单细胞) |

### 4.3 显示配置 (DispConfig)
| 参数 | 默认值 | 说明 |
|------|--------|------|
| fps | 60 | 帧率 |
| scale | 6 | 窗口放大倍数 |

### 4.4 随机种子配置 (RandomConfig)
| 参数 | 默认值 | 说明 |
|------|--------|------|
| seed | 42 | 随机种子，便于复现；设为None表示不固定 |

### 4.5 存档配置 (SaveConfig)
- 自动存档间隔: 5000 tick
- 存档路径: `saves/`
- 格式: txt (细胞数据) + npy (NTs矩阵)

## 5. 错误处理
- 越界移动: 不移动但消耗指令
- 繁殖失败: 无可用空位时失败但消耗指令
- 存档读取失败: 记录日志并抛出异常
- 启动入口错误: 缺失包上下文、命令映射或导入失败时，应修复为可直接执行
- 文档规则冲突: 发现表述矛盾时，应先统一规则再继续执行
- 交互页需求不清: 在规则、输入格式、保存方式未确认前，不进入实现阶段
- 交互参数非法: 输入 DNA/坐标/数量/channel 非法时，阻止提交并记录错误日志
- `spawn` 缺省参数处理: 缺少可选参数时使用默认值，不应抛出 `KeyError`。
- 帧回退失败: 历史帧不足时禁止后退并提示
- restart 失败: 重置过程异常时提示并记录错误日志
- DNA 规则弹窗打开失败: 提示错误并写日志
- DNA 规则弹窗关闭操作失败: 记录错误日志并保留主界面可交互
- DNA 规则弹窗快捷键/按钮触发失败: 记录错误日志并给出提示
- DNA 规则弹窗拖动失败: 拖拽事件异常时记录错误日志并保持弹窗可关闭
- 控制台回退失败: 无历史命令可回退时给出提示，不影响运行
- 控制台复制/粘贴失败: 剪贴板不可用或权限受限时给出提示并写日志
- 方向属性非法: `direction` 非 `w/a/s/d/None` 时拒绝写入并记录日志
- `log5` 失败: 日志文件读取/解析/导出异常时给出提示并写日志，不影响主循环
